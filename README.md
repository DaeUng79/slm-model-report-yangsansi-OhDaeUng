# 2026 AI 챔피언 고급 과정 AI 데이터분석 전문인재 과정

이 저장소는 **실습과제 등록 장소**입니다.

## 과정 소개
기획하는 거점리더를 위한 8주 AI 실전 여정입니다.  
기획자가 그린 그림이 있어야 에이전트의 구현이 시작됩니다.

본 과정은 기술자 양성 과정이 아니라, 공공 현장의 거점리더를 양성하는 과정입니다.  
즉, 자기 부서의 문제를 가장 잘 이해하고 동료에게 AI 활용을 코칭·확산할 수 있는 인력을 목표로 합니다.

수강생은 REST API 내부 구현이나 벡터 DB 알고리즘 세부를 직접 다루지 않지만,  
산출물의 적정성을 검증하고 동료에게 설명할 수 있는 최소 원리를 갖추게 됩니다.

핵심 역량은 다음 한 문장으로 요약됩니다.  
**"어떤 재료를 어떤 순서로 엮으면 이 문제가 풀릴까"를 아키텍처로 그리는 능력**

## 8주 구성
- 1~3회차: AI 도구·서비스 생태계를 지도화하고 워크플로우를 기획
- 4~5회차: 외부망 AI 어시스트로 코드를 받아 내부망에서 실행하는 공공 현실 기반 바이브 코딩
- 6~8회차: 에이전트 도구로 본격 개발(RAG·웹서비스·API·MCP 조합) 후 풀스택 서비스 기획·구축

## 실습과제 등록 안내
- 과제 산출물(HTML/MD/코드)은 본 저장소에 업로드합니다.
- 파일명은 `소속_이름_과제명` 형식을 권장합니다.
- 실습과제 1(SLM 모델 조사 보고서)은 과제 폴더에 등록합니다.
